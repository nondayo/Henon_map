# Quick start guide

#### Install these requirements
- R 3.6+
- shiny
- ggplot2
- dplyr

```
install.packages(c("shiny", "ggplot2", "dplyr"))
```

To use this app, please open your R software and send the following commands.

```
library(shiny)
runGitHub("Henon_map", "nondayo")
```
